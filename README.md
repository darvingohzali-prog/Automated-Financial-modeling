# Automated Financial Modeling

An end-to-end Master Budget Automation project that streamlines financial planning, budgeting, financial reporting, and business performance analysis.

Built with Python for financial modeling and ETL, Excel VBA for spreadsheet automation, and Power BI for executive dashboards. The project automates the complete budgeting workflow from Sales Budget to Budgeted Financial Statements and provides financial ratio analysis, Altman Z-Score, and interactive business performance dashboards.
